# strudel-collab

A collaboration workspace for [Strudel][strudel] music.

This repository is used to share and track the history of tracks.

[strudel]: https://strudel.cc/workshop/getting-started/

## Usage

Open up a `*.strudel` file. Copy its content into [the Strudel REPL][repl]. Enjoy.

[repl]: https://strudel.cc
